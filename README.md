# DataScienceMaster_UOC_Premier_League
Aquest repositori conté la segona pràctica de l'assignatura Tipologia i cicle de vida de les dades, del Màster den Data Science de a UOC.

En el fitxer main es pot trobar el codi de R Markdown executable que retorna el html Notebook amb el codi i els resutlats de la pràctica.

Dins de la carpeta de dades hi ha les dades que s'han utilitzat en format csv i les dades finals després de les transformacions que shan fet al llarg de la pràctica.
